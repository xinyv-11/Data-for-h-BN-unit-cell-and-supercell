# Data-for-h-BN-unit-cell-and-supercell

This compressed archive contains two parts, corresponding to the unit-cell and supercell calculations of bilayer h-BN presented in this work.

Unit cell: Crystal structures (POSCAR) and band information (PROCAR) for monolayer and bilayer h-BN with various stacking configurations (AA,B^BN, AB, B^BB, B^NN), including cases under biaxial tensile strain.

Supercell: Crystal structures (POSCAR) and band information (PROCAR) for twisted bilayer h-BN moiré superlattices (tMSL) at different twist angles (e.g., 3.89° and 5.09°), including strained configurations.
